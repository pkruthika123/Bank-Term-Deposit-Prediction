# Bank Term Deposit Prediction

## 📌 Project Overview
This project analyzes customer data to predict whether a client will subscribe to a bank term deposit.

## 📊 Dataset
- 41,188 records
- 21 features

## 🔍 Key Tasks
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Logistic Regression Modeling

## 📈 Results
- Accuracy: ~91%
- Identified key factors affecting customer decisions

## 🧠 Key Insights
- Call duration strongly impacts subscription
- Previous campaign success improves conversion rate
- Cellular contact is more effective than telephone

## 🛠 Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, Jupyter Notebook

## 📌 Conclusion
This project demonstrates how data analytics and machine learning can help improve marketing strategies by targeting the right customers.
